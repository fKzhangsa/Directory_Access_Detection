# Directory_Access_Detection
# 描述
在windows10中实时检测你指定的文件目录被何种程序访问，并提醒你。
# 安装方式
该脚本依赖于微软的handle工具获得句柄信息，所以要使用该工具请前往https://learn.microsoft.com/zh-cn/sysinternals/downloads/handle
下载hadle，下载完成后将工具添加环境变量。即可使用该脚本。
